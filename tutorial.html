Welcome to Osbourne Explorer. App is designed to position small areas of external raster maps (called overlays once positioned) at google maps.

UI

Buttons:

  1. Go to GPS position
  2. Change google map type (sat/top)
  3. Toggle all overlays transparency
  4. Toggle all overlays visibility

You can pull out the navigation menu by grabbing any of the buttons and dragging it to the left.

Navigation menu:

By touching overlay or marker name you can bring up additional menu which will reveal it's position.
You can drag any overlay or marker object to the left to reveal additional actions.

Positioning

Map positioning done in 6 steps:
  1. Selecting image from iTunes shared folder
  2. Cropping it
  3. Selecting first bind point on raster map
  4. Selecting the same point on google map
  5. Selecting second bind point on raster map
  6. Selecting the same point on google map
  7. Checking & modifying overlay metadata
Transfering & selecting image

You can transfer raster map images (JPEG or PNG) directly in app's documents directory using iTunes.

Afterwards you select image after touching "Add new overlay".

Cropping images

Select area of the image by scrolling and zooming. It's better to use smaller images since app is designed to position small areas.

Selecting bind points

Next thing you do - selecting 2 pairs of binding points. You should select point on external map and select exact same point again, but on internal one. Repeat process one time again.

Memory issues

If you're using big overlays, or using app on older devices, you may encounter memory issues. Tips to prevent/fix them:

  • Use smaller overlays (crop images to smaller sizes)
  • Close application (using app switcher) and start it again

Settings

You can access settings by tapping on "Settings" entry in navigation menu. Several options are presented:

  • Restrict overlay size - default to ON - if checked, overlay size is restricted to prevent memory issues.
  • Clean temp. files - sometimes overlays no longer presented in the app will leave temporary files. Use this option to cleanup them.
